# CRON JOB SETUP INSTRUKCIJE
# 
# Za automatsko pokretanje trading simulatora svakih 30 minuta:
#
# 1. Napravite run_trading.sh izvršnim:
#    chmod +x run_trading.sh
#
# 2. Otvorite crontab editor:
#    crontab -e
#
# 3. Dodajte sledeću liniju (zamenite /full/path/to sa punim putanjom do vašeg projekta):
#    */30 * * * * /full/path/to/stocks/run_trading.sh
#
# ALTERNATIVNO - za različite intervale:
#
# Svakih 15 minuta:
# */15 * * * * /full/path/to/stocks/run_trading.sh
#
# Svakih 1 sat:
# 0 * * * * /full/path/to/stocks/run_trading.sh
#
# Samo radnim danima od 9-17h svakih 30min:
# */30 9-17 * * 1-5 /full/path/to/stocks/run_trading.sh
#
# Provera aktivnih cron job-ova:
# crontab -l
#
# Uklanjanje cron job-a:
# crontab -e (pa obrisati liniju)
#
# NAPOMENE:
# - Cron koristi punu putanju, ne relativnu
# - Logovi se čuvaju u data/cron.log
# - Za debugging, možete promeniti u skripti: python main.py run >> data/debug.log 2>&1
